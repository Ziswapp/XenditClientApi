# XenditClientApi
This is unofficial xendit client api, built with `symfony/http-client`. currently support :

- [x] Virtual account
- [ ] Credit card
- [ ] eWallets (OVO)
- [ ] Retail Outlets (Alfamart Group)

# Install

```
composer require ziswapp/xendit-client-api
```
